SUBGRUP-PROP43.2

Descripcio del directori: Aquest directori conte tots els elements relacionats amb la primera entrega del projecte. 

  DOCS:
    Conte tota la informacio relativa al projecte:
      - Diagrama de Casos d'Us
      - Diagrama del Model Conceptual de Dades
      - Documentacio dels codis font
      - Documentacio amb descripcions, explicacions i justificacions dels casos d'us, el diagrama conceptual,
        l'algorisme, les estructures de dades i els tests unitaris.

  EXE:
    Fitxers executables.

  FONTS:
    Codis de les classes de domini associades a les funcionalitats principals implementades. Inclou tambe els
    tests JUnit i els Drivers amb els jocs de prova.



COMPILAR I EXECUTAR:
  S'ha de compilar i executar des del directori FONTS

    Linux:
    1. Compilar amb la comanda:   make
    2. Executar amb la comanda:   ./scrabble.sh
   	
    Windows:
    1. Compilar amb la comanda:   .\compilar.bat
    2. Executar amb la comanda:   .\scrabble.bat



COMPILAR I EXECUTAR TESTS UNITARIS:

    !! Que s'imprimeixin missatges d'error de Java no vol dir que els tests fallin, nomes cal fixar-se en els missatges de JUnit !!

    Linux:     ./test.sh
    Windows:   .\test.bat


EXECUTAR JOCS DE PROVA:
    Per executar els jocs de prova, executar el driver com s'indica a dalt i premer la opcio pertinent.
    S'obriran dues finestres, una per escollir l'input i la segona per seleccionar l'output esperat.
    El propi driver compararà l'output generat amb l'output esperat.
    Els jocs de prova no inclouen jugar partida ja que l'aleatorietat hi juga un gran paper.
