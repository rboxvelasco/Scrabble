SUBGRUP-PROP43.2

Descripcio del directori: Directori amb codis font. Els subdirectoris segueixen l'estructura de packages.
	
Elements del directori:
    - Carpeta domini: conte els .java de les classes de domini.
    - Carpeta controladors: conte els .java dels controladors.
    - Carpeta persistencia: conte els .java de les classes de persistencia.
    - Carpeta interficie: conte els .java de les classes de presentacio.
    - Carpeta resources: conte els arxius auxiliars per al funcionament de l'aplicacio.
    - Carpeta lib: emmagatzema les llibreries necessaries pel projecte.
    - Carpeta data: s'hi guardaran les dades persistents de l'aplicació.
 

COMPILAR I EXECUTAR:
  S'ha de compilar i executar des del directori FONTS

    Linux:
    1. Compilar amb la comanda:   make
    2. Executar amb la comanda:   ./scrabble.sh
   	
    Windows:
    1. Compilar amb la comanda:   .\compilar.bat
    2. Executar amb la comanda:   .\scrabble.bat
         o premer l'icone "Scrabble - PROP" des de l'explorador d'arxius.


FONTS
│
├── domini // Codis font de la capa de domini.
│    ├── auxiliars
│    ├── jugadors
│    ├── scrabble
│    ├── sessio
│    ├── excepcions
│    └── diccionari
│
├── controladors  // Codis font dels controladors.
├── persistencia  // Codis font de la capa de persistencia.
├── interficie    // Codis font de la capa de presentacio.
├── lib           // Llibreries necessaries per al funcionament de l'aplicacio.
├── resources     // Arxius necessaris pel funcionament de l'aplicacio i per a generar la documentacio Doxygen.
├── data          // Directori que emmagatzema les dades persistents de l'aplicació.
│
├── Makefile             // Makefile per a compilar amb Linux.
├── compilar.bat         // Script per a compilar amb Windows.
├── scrabble.sh          // Script per a executar l'aplicacio amb Linux.
├── scrabble.bat         // Script per a executar l'aplicacio amb Windows.
├── scrabble.vbs         // Script per a executar l'aplicacio amb Windows (sense informació de debugging).
├── Scrabble - PROP.lnk  // Acces directe per a executar l'aplicacio des de l'explorador d'arxius (Windows).
└── index.txt
