Descripció Detallada

Classe que representa la finestra principal del menú de l'aplicació Scrabble.

Aquesta classe mostra el menú principal de l'aplicació, que ofereix les opcions: 1) Iniciar Sessió 2) Crear Perfil 3) Eliminar Perfil
